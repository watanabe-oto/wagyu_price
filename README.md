# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？
- リモートリポジトリとはウェブ上の保存場所のため、ほかの人と共有することができる
- 自身のPC内の保存場所


## プッシュとマージの違いは何でしょうか？
- プッシュとはローカルリポジトリにある変更内容を、リモートリポジトリに送信してアップロードする
- マージとは別のブランチで作業したものをベースとなるブランチに合流させる
## コミットとプッシュの違い
- コミットとは自分のPCの中でのファイルの変更を記録して保存すること
- プッシュとはコミットして記録したものをネット上のGitHubへ送信して同期すること


## コミットのメッセージはどのように書いてあげるのが最適でしょうか？
- 履歴がどんな編集を行ったのかを分かりやすく示す


## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？
- Git Bushのみで完結し、ブランチを合流させる
- プルリクエストでマージとはGitHub上で変更したものを通知し、レビューしてもらい合流させる


## コンフリクトを起こしてしまった場合、どう対処すべきですか？
- 先にマージされた変更内容を取り込む
- 後にマージした変更内容を取り込む
- どちらの変更内容も取り込む
